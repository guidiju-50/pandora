# CONTROL Module

## Descrição
Módulo de backend responsável pelo controle e gerenciamento de dados.

## Tecnologias
- **Linguagem:** Go (Golang)
- **Banco de Dados:** PostgreSQL

## Estrutura
```
CONTROL/
├── cmd/           # Pontos de entrada da aplicação
├── internal/      # Código interno do módulo
├── pkg/           # Pacotes reutilizáveis
├── migrations/    # Migrações do banco de dados
└── README.md
```
