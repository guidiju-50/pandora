# OPERATION Module

## Descrição
Módulo de frontend responsável pela interface de usuário do sistema Pandora.

## Tecnologias
- **Framework:** Vue.js

## Estrutura
```
OPERATION/
├── src/
│   ├── components/    # Componentes Vue
│   ├── views/         # Páginas/Views
│   ├── router/        # Configuração de rotas
│   ├── store/         # Gerenciamento de estado
│   └── assets/        # Recursos estáticos
├── public/
└── README.md
```

## Setup
```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```
