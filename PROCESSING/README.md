# PROCESSING Module

## Descrição
Módulo de backend responsável pelo processamento de sequências biológicas.

## Tecnologias
- **Linguagem:** Go (Golang)

## Estrutura
```
PROCESSING/
├── cmd/           # Pontos de entrada da aplicação
├── internal/      # Código interno do módulo
├── pkg/           # Pacotes reutilizáveis
└── README.md
```

## Setup
```bash
# Inicializar módulo Go
go mod init github.com/guidiju-50/pandora/PROCESSING
```
